## 介绍

DCMC（Direct Connection Minecraft）是一个用Go语言开发的网络通信工具，旨在建立客户端之间的直接连接，而无需依赖中心服务器。该项目实现了NAT穿透技术和房间管理机制，以在分布式环境中实现点对点通信

## 主要功能

- 客户端角色选择（Peer/Host）
- 基于服务器的房间管理
- NAT穿透实现
- 客户端间消息传递
- 命令行界面（CLI）

## 依赖

- Go 1.24.2或更高版本

## 协议

该项目使用自定义数据传输协议，支持整数、字符串、布尔值和浮点数。有关详细协议规范，请参阅`protocol_id.txt`。
